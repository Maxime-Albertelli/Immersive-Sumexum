# Immersive Sumexum

Projet de galerie sur la licence Kingdom Hearts en réalité augmentée

Maxime Albertelli - Henri Gicquel

## Objectif initial

L'utilisateur scanne un QR Code qui affiche le modèle 3D d'une jaquette d'un jeu.
Un bouton "Plus d'information" apparait et redirige vers la fiche d'information de ce jeu.

(voir fichier Figma pour la maquette)

## Mode d'emploi

Imprimer le QR Code se situant dans le dossier /images/pattern/pattern-KingdomHearts.png
- Ouvrir l'application
- Scanner le QR Code avec l'onglet camera et le modèle 3D s'affichera

Pour plus d'information sur un des jeux Kingdom Hearts, cliquez sur le menu déroulant et choisissez-en un.
Sur cette page vous pouvez cliquer sur les onglets "Histoire" et "Dev / sortie" pour les faire apparaitre / disparaitre.
Cliquez sur l'image cdiscount pour être redirigé vers une page d'achat.

## Piste d'amélioration

L'évolution possible serait de créer un modèle 3D par jeux avec chacun leur QR Code. Avoir uniquement un bouton redirigeant vers la page
plus d'information sur l'objet actuellement scanné (cela en liant le QR Code à une valeur du fichier JSON).
